Overview:
 
The project demonstrates the use of popular Python libraries for data analysis and provides meaningful conclusions based on the dataset.


Objectives:

- Load and explore the Student Performance dataset.
- Understand the dataset structure and data types.
- Check for missing values and ensure data quality.
- Perform statistical analysis on student performance.
- Analyze the relationship between study time and final grades.
- Compare academic performance based on gender.
- Create visualizations to better understand the data.

Tools Used:

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

Visualizations:

- Histogram of Final Grades
- Scatter Plot (Study Time vs Final Grade)
- Bar Chart (Average Final Grade by Gender)
